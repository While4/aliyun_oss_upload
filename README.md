# aliyun_oss_upload
> 使用springboot+vue实现的阿里云oss后端签名+前端直传并设置上传回调获取图片大小、宽高、类型、图片地址等信息

> 请将application.yml中的OSS相关配置信息修改为您自己的配置，签名回调地址需要外网可访问
